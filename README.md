# Projeto Banco de Dados Brasileirão 2026

Este projeto foi desenvolvido para representar uma base de dados simples do Campeonato Brasileiro de 2026, usando MySQL.

A ideia principal é organizar informações sobre clubes, estádios, treinadores, jogadores, partidas, temporada, competição e classificação dos times. Além das tabelas principais, o projeto também utiliza procedure, function, trigger, view, roles e permissões de usuários.

## Objetivo do projeto

O objetivo é praticar a criação e manipulação de um banco de dados relacional, aplicando conceitos importantes de SQL, como:

- criação de banco de dados;
- criação de tabelas com chave primária e chave estrangeira;
- inserção de dados;
- relacionamento entre tabelas;
- consultas com `JOIN`;
- criação de `VIEW`;
- criação de `PROCEDURE`;
- criação de `FUNCTION`;
- criação de `TRIGGER`;
- controle de usuários, roles e permissões.
